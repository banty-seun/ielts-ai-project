# Listening Governance Review Cadence

## Quarterly Cadence
1. Review governance KPI trends for the completed quarter.
2. Record action items with owner and due date.
3. Track carry-over actions from the previous quarter.

## Mandatory KPIs
1. Hallucination rejection rate.
2. Override rate.
3. Policy-violation rate.
4. Mean remediation time.

## Output Template
1. Review window (`from`, `to`).
2. KPI summary.
3. Incidents and exception usage.
4. Action items: owner, due date, status.

## Repeated Failure Rule
If any control failure class repeats in two consecutive reviews, backlog reprioritization is mandatory before broader rollout.
